# StarBans Feature Overview

Included systems:
- player bans
- temporary bans
- IP bans
- temporary IP bans
- mutes
- temporary mutes
- kicks with case logging
- notes
- alt-account markers
- IP blacklist
- optional command overrides (`/ban`, `/ipban`, `/mute`, `/kick`, ...)
- case history GUI
- PlaceholderAPI expansion
- Discord webhook logging per action
- optional VPN / proxy detection using ProxyCheck
- JSON / SQLite / MariaDB storage
